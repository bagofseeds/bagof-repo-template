# bagof-repo-template

Template repository for `bagof` Python projects.

This template includes:

- a `pyproject.toml` configured for a `bagof.submodule_name` package
- a `bagof` namespace package under `src/`
- reusable GitHub Actions adapted from
  [`neuroscales/abczarr`](https://github.com/neuroscales/abczarr)

When using the template, replace `submodule_name` with your project-specific
package name.
